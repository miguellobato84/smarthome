# Templates

This folder contains template-based Home Assistant configuration and helper entity definitions.

## Files
- `fan.yaml`: fan-related template/entity config.
- `google_assistant.yaml`: Google Assistant exposure and entity mapping.
- `input_boolean.yaml`: helper input booleans.
- `input_datetime.yaml`: helper datetime inputs.
- `input_number.yaml`: helper numeric inputs.
- `sensor.yaml`: template/statistics sensor definitions.
