# Glossary

Use this file for repository-specific acronyms and internal terms that may appear in prompts, docs, or code.

## Confirmed terms
- `HACS`: Home Assistant Community Store, used for installing custom Lovelace/frontend resources.
- `Lovelace`: Home Assistant dashboard UI system configured here in YAML mode.
- `WallPanel`: Lovelace dashboard/profile intended for wall-mounted tablet-style usage.
- `Zigbee2MQTT`: Bridge service that exposes Zigbee devices over MQTT.
- `MQTT`: Messaging transport used by integrations and devices.
- `AC`: Air conditioning / climate control context in automation names.
- `Federico`: name of the robot vacuum

## Usage notes
- Prefer this glossary over guessing meanings of internal terms.
- Move clarified terms from "Terms requiring clarification" to "Confirmed terms".
