# Community Frontend Resources

This folder contains HACS-managed custom Lovelace card assets.

## Folders
- `lovelace-card-mod/`: card-mod custom styling module.
- `lovelace-mushroom/`: Mushroom card library assets.
- `lovelace-wallpanel/`: wallpanel frontend module.
- `streamline-card/`: streamline-card assets.

Do not edit vendor files here manually; prefer HACS-managed updates.
